Enables smooth scrolling between page sections.  
Sets a clean font, text color, and background for the entire site.

Limits content width for better readability.  
Automatically centers headings, images, and paragraphs.

PART 3: JAVASCRIPT 

1. Waiting for the Page to Load 
The code begins with: 
document.addEventListener("DOMContentLoaded", function () { 
This tells the browser to wait until the entire HTML page has loaded before running the JavaScript. This prevents errors that would occur if the script tried to access elements that had not yet been created. 


2. Active Navigation Link 
The script determines which page the user is currently visiting by examining the page's URL. 
It then loops through all links inside the navigation bar (.main-nav a) and compares each link to the current page. If a match is found, the script adds the class active to that link. 
This allows the current page (such as Contact) to be visually highlighted in the navigation menu so users know where they are on the website. 


3. Accessing the Contact Form 
The script finds the form using: 
const form = document.getElementById("contactForm"); 
It also retrieves the three input fields: 
Full Name  
Email Address  
Message  
In addition, it retrieves the small elements used to display error messages beneath each field. 


4. Clearing Previous Errors 
Before checking the user's input, the script removes any previous error messages and border colors. 
This ensures that old validation messages do not remain visible when the user attempts to submit the form again. 
 

5. Name Validation 
The script checks whether the Full Name field is empty. 
If empty, it displays:  
"Please enter your full name." 
and gives the input field a red border using the input-error class. 
If the field contains text, a green border (input-success) is applied to indicate that the input is valid. 
 

6. Email Validation 
The script checks: 
Whether the email field is empty.  
Whether the email follows a proper format using a regular expression.  
For example: 

‚úî john@gmail.com 

‚úò john@gmail 

‚úò john.com 

If the email format is incorrect, an error message is displayed and the input receives a red border. 

 

7. Message Validation 

The script checks whether: 

The message field is empty.  

The message contains at least ten characters.  

If the message is too short, the user sees: 

"Message must contain at least 10 characters." 

and the textarea receives a red border. 

 

8. Preventing Automatic Submission 

When the user clicks the Send Message button, the script uses: 

e.preventDefault(); 

This prevents the browser from refreshing the page or opening an email application. 

Instead, the form remains on the same page while validation is performed. 

 

9. Determining Whether the Form is Valid 

The variable: 

let valid = true; 

acts like a switch. 

If any field contains invalid information, valid becomes false.  

If every field passes validation, valid remains true.  

 

10. Displaying a Success Message 

If all three fields contain valid information, the following code executes: 

alert("‚úì Information submitted successfully!"); 

This displays a popup notification informing the user that the form has been successfully submitted. 

 

11. Resetting the Form 

After the success message appears, the script clears all the input fields: 

form.reset(); 

This prepares the form for another submission. 

 

12. Removing Green Borders 

Since the fields have been emptied, the script removes the green success borders from all inputs so that the form returns to its original appearance. 

 

Overall Flow of Execution 

The page loads.  

JavaScript waits for the HTML to finish loading.  

The current navigation link is highlighted.  

The user enters information into the contact form.  

The user clicks Send Message.  

JavaScript prevents the page from submitting.  

The Full Name, Email, and Message are checked.  

Error messages are displayed if any information is invalid.  

Valid fields are highlighted in green.  

If all fields are valid, a popup appears saying:  

‚úì Information submitted successfully! 

The form is cleared and reset for future use.
